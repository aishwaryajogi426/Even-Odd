# Sum of square of first n numbers

## Description
A simple Java program that gives the sum of the square of first n numbers starting from 0.

## Features
- Takes user input
- Displays result on console

## Technologies Used
- Java

## How to Run
1. Compile the program using `javac SqaureN.java`
2. Run the program using `java SquareN`

## Concepts Used
- For loop

## Output
Displays the sum of the square of first n numbers
