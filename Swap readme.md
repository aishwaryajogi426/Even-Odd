# Swapping of two numbers

## Description
A simple Java program that swaps the values of an int variable without using a temporary variable

## Features
- Takes user input
- Displays result on console

## Technologies Used
- Java

## How to Run
1. Compile the program using `javac Swap.java`
2. Run the program using `java Swap`

## Concepts Used
- Logic

## Output
Displays the interchanged values of two variable along with orirginal values
